1. To run the entire process end to end, run: sh create_upload_execute.sh inside the respective folder.

This will create server instance, upload files and execute MapReduce.

2. If you want to test for another file, change name in config.ini, ensure that upload code for that file is in upload_files.sh 
and run upload_files.sh and then execute.sh

3. To delete everything once testing is done, run delete.sh
